# PatrickZhang-SWE.github.io

My personal website.

## Create a new post

```shell
jekyll post "the is a new post"
```

## Install required packages

```shell
bundle install
```

## Run server locally

```shell
bundle exec jekyll serve --host=0.0.0.0 //default port is 4000
```

if host is omitted, 127.0.0.1 will be used.
